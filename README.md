# customer_behavior_analysis
This project analyzes customer shopping behavior using transactional data from 3,900  purchases across various product categories. The goal is to uncover insights into spending  patterns, customer segments, product preferences, and subscription behavior to guide strategic  business decisions. 



## Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases. The objective is to identify customer spending patterns, product preferences, subscription trends, and business opportunities through data analysis and visualization.

## Tools & Technologies

* Python (Pandas, NumPy)
* PostgreSQL
* Power BI

## Project Workflow

1. Data Cleaning & Preprocessing in Python
2. Data Storage and Analysis in PostgreSQL
3. Business Insights Generation using SQL
4. Interactive Dashboard Development in Power BI

## Key Analyses

* Revenue by Gender
* High-Spending Discount Users
* Top Rated Products
* Shipping Type Comparison
* Subscriber vs Non-Subscriber Spending
* Customer Segmentation
* Top Products by Category
* Revenue by Age Group
* Repeat Buyer Subscription Analysis

## Dashboard Features

* Customer Demographics Overview
* Revenue & Purchase Trends
* Product Performance Insights
* Subscription & Loyalty Analysis
* Interactive Filters and Visualizations

## Business Recommendations

* Strengthen customer loyalty programs
* Increase subscription adoption
* Optimize discount strategies
* Promote top-performing products
* Target high-value customer segments

## Dataset

Customer shopping behavior dataset containing 3,900 transactions and 18 features covering demographics, purchases, reviews, discounts, subscriptions, and shopping preferences.
